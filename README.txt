声明：
1、本站基于bootstrap进行开发
2、本站整体设计风格为扁平风，主色调为绿色
3、本站数据均来源华软软件系官网，即本站内容观点不代表本人观点

文件分析：
1、base.css、base.js为公共类，即共用类，该类定义了所有页面的导航栏以及页脚的样式和交互效果
2、其余样式分别对应各自文件名

代码分析：
1、所以轮播图均为bootstrap框架组件
2、本站所有便签均遵循 H5新标准的便签语义化
3、本站所有动画效果均基于 CSS3的 transition属性 进行延伸扩展，极少书写JS实现动画
4、地理定位时调用了百度API，故准确度不是特别高 ps：因为谷歌在大陆被被封杀了...我也很绝望...
